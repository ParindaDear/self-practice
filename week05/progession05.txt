findIndex() ต่างกับ indexOf() ยังไง (ใช้อันไหนก็ได้)
- indexOf() จะ return first index ของ element ที่ให้ไว้ โดยถ้าไม่เจอจะ -1
  มันต้องการ parameter ที่เป็น string
- findIndex() ไว้หา index ของ first element
  parameter มันต้องการ callback func ที่ทำหน้าที่ให้ condition
- ความเหมือนคือ output  เเต่ความต่างคือ input parameter 

Tree Shaking คือ remove code ที่เราไม่ได้ใช้ออก 
โดยจะดูความผูกพันธ์ตรง module import, export 
การประกาศมี 2 ระดับคือ 
1. Global คือทุก file สามารถอ้างถึง module / file นี้ได้ โดยไม่ต้อง import 
2. Local อ้างไว้ที่ไหนใช้ได้ที่นั่น ไฟล์อื่นใช้ไม่ได้ (ทำให้ Tree Shaking ดีสุด) 
   จะใช้ไฟล์ไหนต้อง import เพราะถ้าอันไหนไม่มี import มันจะลบทิ้งทั้งหมด

๋JS Module มี 2 syntax
- CommonJs ใช้ syntax ของ Node.js 
- ES Modules ใช้กับ browser 
- เเต่ปัจจุบันมีตัวช่วยละว่าจะข้าม syntax ต้องทำไงโดยเปลี่ยนนามสกุลจาก .js เป็น .mjs ทำให้ไม่ต้องเปลี่ยน syntax ไปมา
