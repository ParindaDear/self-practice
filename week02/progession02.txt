เริ่มเรียนเรื่อง Object 
- key ซ้ำได้ เเต่ก็ไม่ควร!!! เพราถ้า key ซ้ำมันจะเอา value ตัวสุดท้าย
- การ + มี 2 ความหมาย ดูจากตัว operand 
    + เลข ต่อเมื่อทั้ง 2 ฝั่งเป็นเลข
    + string ต่อเมื่อฝั่งใดผั่งนึงเป็นข้อความ
- Property names are usually Strings or can also be Symbols
    Symbol คือ unique key 
    ปล. string กับ symbols ไม่เหมือนกันนะ
- JS เป็น implicit คือ ไม่ต้องประกาศ class ก่อนก็สร้าง object ได้
- การสร้าง Object มี 4 วิธีหลักๆคือ
    1. Object literals 
        ข้อดีเขียนได้ดี เขียนได้ไว
        ข้อเสียคือถ้าสร้างหลายอันต้อง repeat code เอง
    2. Object Constructor 
        ข้อดีคือมันเริ่มเป็นลูกครึ่งละเขียนไม่ยากเหมือน class ไม่ต้อง repeat บ่อยๆให้ให้ new operator เลย
        ข้อเสียคือดูยากเพราะมันหน้าตาแอบคล้าย func ทั่วไป, การใช้ new บางทีอาจสับสน
    3. ES6 Classes ถ้าต้องการสร้างหลาย object เเละสามารถใช้ function เดียวกันแชร์ด้วยกัน
